## nestjs starter


## Description

A template to get a nestjs server up and running

### Author
- chenc
- double_cl@163.com

## Preset
- winston log
- jwt auth
- env or dotEnv file
- typeorm (mysql)
- validate (class-validator & class-transformer)

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
